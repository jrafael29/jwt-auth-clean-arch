Ideia de portfolio

API para gerenciamento de geolocalizações.
Utilizando SSE, Clean Arch

Tecnologias a serem exercitadas.

- Docker
- NodeJS
- MongoDB
- Leaflet
- Redis
- Server-Sent-Events
- JWT

# Features

- Usuario poderá se cadastrar
- Usuario poderá se autenticar 
- Usuario poderá visualizar outros usuarios em um raio de X KM 
- Usuario poderá ter X endereços cadastrados, e apenas 1 em USO. 
- O endereço em uso, será o exibido para outros usuarios.










# Rotas não autenticadas
- Visualizar localizações

# Rotas autenticadas
- Cadastro de localização ( para o usuario ) 
- visualizar localizações ( para o usuario )
